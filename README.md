## 🚀 Projetos em Destaque

### 🤟 [LIBRAS Translator](https://github.com/Defendi22/LIBRAS-Translator)
> Tradutor do alfabeto manual de Libras em tempo real, direto no navegador via webcam — sem instalação, sem plugins.

- **ML com 99,94% de acurácia** usando Ensemble (Random Forest + KNN + SVM)
- Treinado com **24.000 amostras** de gestos
- Comunicação via **WebSocket** de baixa latência
- Deploy automatizado com **CI/CD** (GitHub Actions + Render)
- Containerizado com **Docker**

`Python` `FastAPI` `MediaPipe` `scikit-learn` `Docker` `GitHub Actions`

### 📊 [SUS Brasil — Dashboard de Mortalidade](https://github.com/Defendi22/SUS-Brasil)
> Dashboard interativo full stack para análise dos dados do Sistema de Informação sobre Mortalidade (SIM) do Ministério da Saúde.

- **6 KPIs** + linha do tempo + pirâmide etária + mapa por estado
- Filtros interativos por ano, sexo, estado e tipo de óbito
- Extrator de dados via **API pública do Ministério da Saúde** (`Python`)
- Processamento **100% local** — nenhum dado é enviado a servidores externos

`Python` `JavaScript` `Chart.js` `HTML5` `CSS3` `REST API`
